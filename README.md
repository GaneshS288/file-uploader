# file-uploader

This is a simple GDrive/OneDrive clone

## How to use

You need to have a postgres database and a supabase project set up. Follow these steps after you have them

- run npm install to download node packages.
- create a bucket in your supabase account. You must create it yourself this project will not create it for you
- set up enviroment varibales for supabase, postgres DB and Session.
- you sould be good to go now. Do npm run start to launch the project.
